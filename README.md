# Forest
Forest专注森林无限金币脚本

## 声明  
仅供学习，不得用于其他用途  

## 原理  
通过向过去的时间段种植树木（完成学习的时间）来获取金币。

## 脚本  
* forest.py  
获取金币的脚本  
* plant.py（可忽略）  
显示植物名称及对应价格的脚本  

## 参数修改  
* data.txt(可忽略)  
专注所处的周期，用于判断是否种植了植物  
* header.txt(必填)  
填入抓包获取到的header  
* forest.py（自行修改，已有默认配置）
